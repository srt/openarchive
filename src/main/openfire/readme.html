${pom.name} Readme

Overview

Server side message archive for the Openfire XMPP server.

Supports XEP-0136 automated archiving and message retrieval. Support for preferences and manual archiving will follow later.

Limitations

Open Archive currently does not support group chat (muc) messages.

The only databases currently supported out of the box are MySQL and HSQLDB though it is rather easy to port the SQL scripts to other databases.
